Section 4.4 Section 3.9 - Linear Approximation

Worksheet 4.4.1 Worksheet

1.
Use the tangent line approximation of \(f(x)=\ln(x)\) at \(x=1\) to approximate the value \(\ln(1.1)\text{.}\) Solution
Using tangent line approximation at \(x=1\) we see that \(f(1)=\ln(1)=0\text{,}\) \(f'(x) = \frac 1x\text{,}\) and \(f'(1) = \frac 11 = 1\text{,}\) so the tangent line is
\begin{equation*} y = f(1)+f'(1)(x-1) = 0 + 1(x-1) = x-1 \end{equation*}
Using tangent line approximation we see that
\begin{equation*} \ln(1.1) = f(1.1) \approx 1.1-1 = 0.1 \end{equation*}
2.
Let \(f(A)\) represent the price of building a house of area \(A\) ft\(^2\) in Bob the Builder's subdivision. To buy a 2100 ft\(^2\) house it costs $200,000. If \(f'(2100)=90\text{,}\) estimate the cost of a 2150 ft\(^2\) house using tangent line approximation. Solution
Using tangent line approximation we get
\begin{equation*} f(A)\approx f(2100)+f'(2100)(A-2100) = 200000+90(A-2100) \end{equation*}
\begin{equation*} f(2150)\approx 200000+90(2150-2100) = 204500\text{.} \end{equation*}
The bigger house should cost approximately $204,500.
