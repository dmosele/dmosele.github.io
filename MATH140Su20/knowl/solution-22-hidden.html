Using the quotient rule, we get
\begin{equation*} \frac{d}{dx}\left( \tan x \right) = \frac{d}{dx}\left( \frac{\sin x}{\cos x} \right) = \frac{\cos(x)\cos(x)-\sin(x)(-\sin(x))}{(\cos(x))^2} \end{equation*}
\begin{equation*} = \frac{(\cos(x))^2+(\sin(x))^2}{(\cos(x))^2} = \frac{1}{(\cos(x))^2} \end{equation*}
The last equality comes from the Pythagorean Identity.
