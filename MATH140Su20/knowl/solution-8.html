Solution 2.1.1.1.1.
We have three approaches to this, Left, right and average.
\begin{equation*} \text{(Left) } \frac{f(2)-f(1.5)}{2-1.5} = \frac{2.3-2.0}{.5)= 0.6 \text{ in/sec} \end{equation*}
\begin{equation*} \text{(Right) } \frac{f(2.5)-f(2)}{2.5-2} = \frac{2.4-2.3}{.5)= 0.2 \text{ in/sec} \end{equation*}
\begin{equation*} \text{(Average) } \frac{.6+.2}{2} = 0.4 \text{ in/sec} \end{equation*}
