Section 2.2 Domain and Range

Worksheet 2.2.1 Worksheet

1.
Compute the domain of the following functions
  1. \begin{equation*} f(x) = \sqrt{2x-5} \end{equation*}
  2. \begin{equation*} g(x) = \frac{1}{x^2-1} \end{equation*}
Solution
  1. We need \(2x-5\geq 0\) so that the value under the square root is nonnegative. This means that \(2x\geq5\) and hence \(x\geq2.5\) is the domain of the function \(f\)
  2. We need \(x^2-1\neq 0\text{,}\) so we should ask when it is zero. It is zero when \(x^2=1\) or \(x=\pm 1\text{.}\) Thus the domain is all real numbers except for \(1\) or \(-1\)
2.
For the following questions use the function \(y=f(x)\) graphed below.

  1. Compute the domain and range of \(f\text{.}\)
  2. Compute \(f(2)\text{.}\)
  3. Find values of \(x\) so that \(f(x)=2\text{.}\)
Solution
  1. The domain is \(-4.5\leq x \leq \) because this represents the \(x\) values present in the graph. The range is \(-2\leq y \leq 2\) because this represents the \(y\) values present in the graph.
  2. \(f(2)=-1\) as this is the \(y\) coordinate that corresponds to the \(x\) coordinate 2.
  3. From the graph it looks like \(x=-1, x=3\) are the values with \(f(x)=2\text{.}\) These can be checked by computing \(f(-1),f(3)\) with Desmos.
