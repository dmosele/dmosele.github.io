Solution 6.3.1.2.1.

There are several approaches here, but let's start with some algebra so that we can get rid of the fraction.

\begin{equation*} y^2(xy-4) = x^2 \end{equation*}
\begin{equation*} xy^3-4y^2 = x^2 \end{equation*}

Taking the derivative of both sides we get

\begin{equation*} x(3y^2)\frac{dy}{dx} + y^3 - 8y\frac{dy}{dx} = 2x \end{equation*}

Solving for \(\frac{dy}{dx}\) we get

\begin{equation*} \frac{dy}{dx}(3xy^2-8y) = 2x-y^3 \end{equation*}
\begin{equation*} \frac{dy}{dx} = \frac{2x-y^3}{3xy^2-8y} \end{equation*}

We want to compute the tangent line at \((4,2)\text{,}\) so we plug in 4 for \(x\) and 2 for \(y\text{.}\)

\begin{equation*} \left. \frac{dy}{dx} \right\vert_{(4,2)} = \frac{2(4)-2^3}{3(4)(2^2)-8(2)} = 0 \end{equation*}

Since the slope is zero, the tangent line is \(y=2\) as that is the \(y\)-coordinate of the point.
