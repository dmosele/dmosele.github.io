Section 6.1 Section 3.5 - Trigonometric Functions

Worksheet 6.1.1 Worksheet

1.

So we know that

\begin{equation*} \frac{d}{dx} ( \sin x ) = \cos x \hspace{.2in} \text{ and } \hspace{.2in} \frac{d}{dx} (\cos x) = - \sin x \end{equation*}

and we know that

\begin{equation*} \tan x = \frac{\sin x}{\cos x} . \end{equation*}

Compute the derivative of \(\tan x\text{.}\)

Solution

Using the quotient rule, we get

\begin{equation*} \frac{d}{dx}\left( \tan x \right) = \frac{d}{dx}\left( \frac{\sin x}{\cos x} \right) = \frac{\cos(x)\cos(x)-\sin(x)(-\sin(x))}{(\cos(x))^2} \end{equation*}
\begin{equation*} = \frac{(\cos(x))^2+(\sin(x))^2}{(\cos(x))^2} = \frac{1}{(\cos(x))^2} \end{equation*}

The last equality comes from the Pythagorean Identity.

2.

 1  The voltage, \(V\text{,}\) in volts, in an electrical outlet is given as a function of time, \(t\text{,}\) in second by the function

\begin{equation*} V = 156 \cos(120 \pi t) \end{equation*}
From problem 62 in section 3.5
  1. Give an expression for the rate of change of voltage with respect to time.
  2. What is the maximum rate of change?
Solution
  1. \(\displaystyle \frac{dV}{dt} = 156(-\sin(120\pi t))(120\pi)\)
  2. The maximum that \(-\sin(120 \pi t)\) can be is 1 as -1 is the minimum of the sine function. Thus, the maximum of the derivative here is \(156\cdot 120\pi\text{.}\)
3.

Compute the following derivatives:

  1. \(\displaystyle \displaystyle \frac{d}{dx} (\sin(x^2))\)
  2. \(\displaystyle \displaystyle\frac{d}{dx} \left( \frac{3\cos(x)}{x^2-7}\right)\)
  3. \(\displaystyle\frac{d}{dx} \left( \frac{\cos(x)}{\sin(x)} \right)\) (p.s. Do you know a name for this function?)
  4. \(\displaystyle \displaystyle\frac{d}{dx} \left( e^{\sin(x)} \right)\)
Solution
  1. \(\displaystyle \displaystyle \frac{d}{dx} (\sin(x^2))= \cos(x^2)\cdot (2x)\)
  2. \(\displaystyle \displaystyle\frac{d}{dx} \left( \frac{3\cos(x)}{x^2-7}\right)= \frac{(x^2-7)(3(-\sin(x)))-(3\cos(x))(2x)}{(x^2-7)^2}\)
  3. \(\displaystyle\frac{d}{dx} \left( \frac{\cos(x)}{\sin(x)} \right) = \frac{\sin(x)(-\sin(x))-(\cos(x))(\cos(x))}{(\sin(x))^2} = \frac{-1}{(\sin(x))^2} \) (the original function is cotangent)
  4. \(\displaystyle \displaystyle\frac{d}{dx} \left( e^{\sin(x)} \right) = e^{\sin(x)}\cos(x)\)
