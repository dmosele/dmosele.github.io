Solution 12.2.1.2.1.
  1. \begin{equation*} a(t) = -32 \end{equation*}
    \begin{equation*} v(t) = -32t+64 \end{equation*}
    \begin{equation*} h(t) = -16t^2+64t+45 \end{equation*}

  2. Setting the derivative (velocity) to zero we get

    \begin{equation*} 0 = -32t+64 \Rightarrow t=2 \end{equation*}

    We know it is a maximum by the second derivative test as \(a(2)=-32\text{.}\)

  3. Plugging the time \(t=2\) back into the height function we get
    \begin{equation*} h(2) = -16(2)^2+64(2)+45 = 109\text{ feet} \end{equation*}
