Section 4.3 Section 2.4 - Preview of Transformations: Shifts

Worksheet 4.3.1 Worksheet

1.

The graph the function \(f(x)= x^2\) is drawn on the on the following axes.

  1. On the same axes, graph \(f(x)+2\text{.}\) (What is \(f(x)+2\) you say? \(f(x)+2 = x^2+2\text{.}\))
  2. On the same axes, graph \(f(x)-2\text{.}\)
  3. Make a conjecture (educated guess) as to what adding or subtracting a positive constant from a function does to its graph.
Solution

  1. As can be seen from the graph above, this results in a shift up by 2.
  2. As can be seen from the graph above, this results in a shift down by 2.
  3. Adding a positive constant shifts up by as many units, whereas subtracting a positive constant shifts the graph downward.
2.

The graph the function \(g(x)= \vert x \vert\) is drawn on the on the following axes.

  1. On the same axes, graph \(g(x+2)\text{.}\)
  2. On the same axes, graph \(g(x-2)\text{.}\)
  3. Make a conjecture (educated guess) as to what adding or subtracting a positive constant from the input of a function does to its graph.
Solution

  1. We see that this results in a shift left by 2 units.
  2. We see that this results in a shift right by 2 units.
  3. Adding to the input shifts the function's graph left, whereas subtracting shifts it right.
