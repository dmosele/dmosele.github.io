Section 4.2 Exponential Functions #2

Worksheet 4.2.1 Worksheet

1.
  1. Write a formula for population \(P\) of manatees as a linear function of time \(t\) since 2010.
  2. Write a formula for population \(P\) of manatees as an exponential function of time \(t\) since 2010.
  3. What is predicted for the population in the year 2020?
Solution

  1. Computing the slope we get

    \begin{equation*} m = \frac{1600-1200}{8-0} = 50 \text{ manatees/year} \end{equation*}

    Putting it all together, we get that

    \begin{equation*} P = 1200+50t \end{equation*}

  2. To start with, we know that it is of the form \(P=ab^t\) where in this case \(a=1200\text{.}\) Using the fact that when \(t=8\text{,}\) \(P=1600\) we see that

    \begin{equation*} 1600=1200b^8 \quad \Rightarrow \quad \frac{1600}{1200}=b^8 \quad \Rightarrow \quad b = \sqrt[8]{\frac 43} \end{equation*}
    \begin{equation*} P = 1200\left( \sqrt[8]{\frac 43} \right)^t \end{equation*}

  3. Using the linear model, we get

    \begin{equation*} P = 1200+50(10) = 1700\text{ manatees} \end{equation*}

    Using the exponential model, we get

    \begin{equation*} P = 1200\left( \sqrt[8]{\frac 43} \right)^{10} \approx 1719 \text{ manatees} \end{equation*}
2.
  1. Assuming that the car loses its value exponentially, find a formula for value \(V\) as a function of time since it was purchased.

  2. When will the car be worth $5000?

Solution

  1. The starting value is $20000, so

    \begin{equation*} V = 20000b^t \end{equation*}

    Since the car is worth $8000 when \(t=5\text{,}\)

    \begin{equation*} 8000 = 20000b^5 \quad \Rightarrow \quad b^5 = \frac{8000}{20000} = .4 \quad \Rightarrow \quad b = \sqrt[5]{.4} \end{equation*}

    Thus, the function is

    \begin{equation*} V = 20000\left(\sqrt[5]{.4}\right)^t . \end{equation*}

  2. Graphing the exponential function and a horizontal line at 5000, we see that it takes approximately 7.565 years for the value of the car to reduce to $5000.
