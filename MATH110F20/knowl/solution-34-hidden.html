The formula for the account balance is

\begin{equation*} B = 1200e^{0.05t} \end{equation*}

We want to know when it reaches $1600, so we set \(B\) to 1600 and solve for \(t\text{.}\)

\begin{equation*} 1600 = 1200 e^{0.05t} \quad \Rightarrow \quad \frac{1600}{1200} = e^{0.05t} \quad \Rightarrow \quad \ln\left(\frac{1600}{1200}\right) = \ln(e^{0.05t})=0.05t \end{equation*}
\begin{equation*} t = \frac{\ln\left(\frac{1600}{1200}\right) }{0.05} \end{equation*}
