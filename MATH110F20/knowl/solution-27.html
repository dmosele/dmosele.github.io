Solution 6.2.1.1.1.

  1. Computing the slope we get

    \begin{equation*} m = \frac{1600-1200}{8-0} = 50 \text{ manatees/year} \end{equation*}

    Putting it all together, we get that

    \begin{equation*} P = 1200+50t \end{equation*}

  2. To start with, we know that it is of the form \(P=ab^t\) where in this case \(a=1200\text{.}\) Using the fact that when \(t=8\text{,}\) \(P=1600\) we see that

    \begin{equation*} 1600=1200b^8 \quad \Rightarrow \quad \frac{1600}{1200}=b^8 \quad \Rightarrow \quad b = \sqrt[8]{\frac 43} \end{equation*}
    \begin{equation*} P = 1200\left( \sqrt[8]{\frac 43} \right)^t \end{equation*}

  3. Using the linear model, we get

    \begin{equation*} P = 1200+50(10) = 1700\text{ manatees} \end{equation*}

    Using the exponential model, we get

    \begin{equation*} P = 1200\left( \sqrt[8]{\frac 43} \right)^{10} \approx 1719 \text{ manatees} \end{equation*}
